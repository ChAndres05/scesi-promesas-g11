MÓDULO WEB

Seguridad Web

La seguridad web ayuda a proteger la información del usuario cuando navega, inicia sesión, llena formularios o descarga contenido de internet.

🔐

HTTPS

Protege la información enviada entre el usuario y el sitio web mediante una conexión segura.

🍪

Cookies

Guardan datos del usuario, como sesiones o preferencias. Deben usarse correctamente para evitar riesgos de privacidad.

🧾

Formularios seguros

Ayudan a evitar el robo de datos personales mediante validaciones, cifrado y buenas prácticas.

Riesgos comunes

  • Enlaces sospechosos enviados por correo, redes sociales o mensajes.
  • Descargas inseguras desde páginas desconocidas.
  • Sitios web falsos que imitan bancos, tiendas o plataformas conocidas.
  • Formularios que solicitan datos sensibles sin una conexión segura.
BUENAS PRÁCTICAS

Prevención

Para navegar con mayor seguridad, es importante reconocer señales de riesgo y aplicar hábitos responsables.

  • ✅ No hacer clic en enlaces desconocidos o sospechosos.
  • ✅ No descargar archivos desde páginas poco confiables.
  • ✅ Usar sitios web con HTTPS antes de ingresar datos personales.
  • ✅ Mantener actualizado el navegador, sistema operativo y antivirus.
  • ✅ Verificar la dirección del sitio antes de iniciar sesión.
