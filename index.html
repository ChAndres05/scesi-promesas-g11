Amenazas y ataques comunes

Amenazas y ataques comunes

Descubre las técnicas más frecuentes en ciberseguridad, aprende con ejemplos reales y conoce cómo protegerte de cada amenaza.

🦠

Malware

Software malicioso diseñado para dañar, espiar o robar datos. Incluye virus, troyanos y gusanos.

  • Infecta dispositivos.
  • Puede robar contraseñas.
  • Permite acceso remoto no autorizado.
Ejemplo: un programa gratis descargado de una web insegura instala un troyano que roba datos bancarios.
🎣

Phishing

Ataque que usa mensajes falsos para engañar al usuario y obtener credenciales, datos personales o acceso a cuentas.

  • Correos o mensajes engañosos.
  • Enlaces a páginas falsas.
  • Urgencia para que actúes rápido.
Ejemplo: recibes un correo que parece de tu banco pidiéndote actualizar tu contraseña en una web falsa.
💸

Ransomware

Secuestra archivos o sistemas y exige un rescate para liberarlos. Puede paralizar empresas y organizaciones.

  • Cifra archivos.
  • Exige pago en criptomonedas.
  • No garantiza recuperación completa.
Ejemplo: al abrir un adjunto comprometido, tu equipo queda cifrado y aparece una nota de rescate.
🕵️

Spyware

Software que espía tu actividad sin permiso para robar información, historial, contraseñas o datos sensibles.

  • Monitorea pulsaciones.
  • Registra sitios visitados.
  • Envía datos a terceros.
Ejemplo: una app aparentemente útil instala un spyware que captura mensajes y credenciales.
🔓

Fuerza bruta

Técnica que prueba muchas combinaciones de contraseñas hasta encontrar la correcta. Funciona mejor cuando las claves son débiles.

  • Ataca cuentas con contraseñas simples.
  • Usa diccionarios automáticos.
  • Es común en servicios sin doble factor.
Ejemplo: un atacante prueba varias contraseñas hasta adivinar tu cuenta de correo porque usas una clave común.

Ingeniería social

Estrategias para manipular personas, no sistemas. El objetivo es que el usuario entregue información o acceso sin sospechar.

📧

Suplantación de identidad

Fingir ser una entidad confiable para pedir datos o permisos.

📱

Mensajes falsos

Envían links o archivos maliciosos haciéndose pasar por servicio técnico o un colega.

👥

Urgencia falsa

Presionan con falso apuro para que tomes decisiones sin verificación.

MÓDULO WEB

Seguridad Web

La seguridad web ayuda a proteger la información del usuario cuando navega, inicia sesión, llena formularios o descarga contenido de internet.

🔐

HTTPS

Protege la información enviada entre el usuario y el sitio web mediante una conexión segura.

🍪

Cookies

Guardan datos del usuario, como sesiones o preferencias. Deben usarse correctamente para evitar riesgos de privacidad.

🧾

Formularios seguros

Ayudan a evitar el robo de datos personales mediante validaciones, cifrado y buenas prácticas.

Riesgos comunes

  • Enlaces sospechosos enviados por correo, redes sociales o mensajes.
  • Descargas inseguras desde páginas desconocidas.
  • Sitios web falsos que imitan bancos, tiendas o plataformas conocidas.
  • Formularios que solicitan datos sensibles sin una conexión segura.
BUENAS PRÁCTICAS

Prevención

Para navegar con mayor seguridad, es importante reconocer señales de riesgo y aplicar hábitos responsables.

  • ✅ No hacer clic en enlaces desconocidos o sospechosos.
  • ✅ No descargar archivos desde páginas poco confiables.
  • ✅ Usar sitios web con HTTPS antes de ingresar datos personales.
  • ✅ Mantener actualizado el navegador, sistema operativo y antivirus.
  • ✅ Verificar la dirección del sitio antes de iniciar sesión.
