Seguridad Web

HTTPS: Protege la información entre el usuario y el sitio.

Cookies: Guardan datos del usuario.

Formularios seguros: Evitan robo de datos.

Riesgos

  • Enlaces sospechosos
  • Descargas inseguras

Prevención

  • No hacer clic en enlaces desconocidos
  • No descargar archivos sospechosos
  • Usar sitios con HTTPS
  • Actualizar antivirus
