# CSESI Promesas G11

Proyecto grupal desarrollado para practicar Git, GitHub, ramas, commits, Pull Requests, reviews y buenas prácticas de trabajo colaborativo.

## Tema del proyecto

Este proyecto es un sitio web educativo sobre ciberseguridad. Su objetivo es explicar de forma clara y visual conceptos básicos como amenazas comunes, protección digital, seguridad web y prevención.

## Objetivos

- Practicar el uso de Git y GitHub en equipo.
- Aplicar Gitflow básico usando `main`, `develop` y ramas `feature`.
- Crear commits pequeños, claros y con buenas prácticas.
- Trabajar mediante Pull Requests.
- Presentar un sitio web informativo, ordenado y responsive.

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript
- Git
- GitHub
